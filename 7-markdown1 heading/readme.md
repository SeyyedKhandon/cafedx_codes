# CH2: Bold & italic

---------- Bold ----------

This is **Bold** text.
This is B**ol**d text.

This is __Alternative Bold__ text.

This is <strong>Bold</strong>.	

---------- _Italic_ ----------

This is *Italic*. 

This is _Alternative Italic_ text.


This is <em>Italic</em>. 

---------- Bold ----------
